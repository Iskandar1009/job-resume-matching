# AI Resume Matcher

AI Resume Matcher — веб-приложение на FastAPI, которое автоматически сопоставляет резюме и вакансии.  
Система анализирует PDF-файлы и вычисляет процент совпадения между кандидатами и вакансиями на основе эмбеддингов модели `Qwen/Qwen3-Embedding-0.6B`.

## Возможности

- Семантическое сравнение текста (а не просто поиск по ключевым словам)
- Раздельная оценка по секциям: должность, навыки, опыт, образование  
- Кэширование текстов и эмбеддингов для ускорения повторных сравнений  
- Пакетная обработка нескольких резюме и вакансий

---

## Запуск проекта

Выполните команду:

```bash
docker-compose up --build -d
```

После этого сервис будет запущен на:

```bash
{{BASE_URL}}:8080
```

## API
Приложение также предоставляет REST API для интеграции со сторонними системами.
Эндпоинт:
```bash
POST /match/
```
Тело запроса (Form Data):
- jobs — список PDF-файлов с описаниями вакансий
- resumes — список PDF-файлов с резюме кандидатов
Ответ содержит JSON с процентами совпадения и пояснениями для каждой пары "резюме — вакансия".
Более подробная документация доступна в Swagger:
```bash
{{BASE_URL}}:8080/docs
```
